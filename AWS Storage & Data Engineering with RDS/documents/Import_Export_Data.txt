3. DIFFERENT METHODS FOR BULK IMPORT / EXPORT

Several methods exist for importing and exporting data in RDS:

1. Python-based Import/Export (Used in Project)

- Tools: Python, Pandas, pyodbc
- Advantages: Easy to automate, allows preprocessing and data transformation.
- Use Case: Medium to large datasets for cloud data pipelines.

2. SQL Server Import and Export Wizard (SSMS)

- Visual tool in SSMS for importing CSV/Excel data.
- Advantages: User-friendly, no code required.
- Limitations: Less suitable for automated pipelines.

3. BULK INSERT Command

- Native SQL Server command for loading large CSV files directly.
- Advantages: Very fast for large datasets.
- Use Case: Enterprise-scale batch ingestion.

4. AWS Data Migration Service (DMS)

- Managed service for migrating or replicating data between databases.
- Advantages: Scalable, supports continuous replication.
- Use Case: Full database migration or high-volume pipelines.
